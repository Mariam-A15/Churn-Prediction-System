# Churn-Prediction-System
A full-stack machine learning web application that predicts the likelihood of customer turnover using Polynomial Logistic Regression.

Features
Predictive Model: Uses a Logistic Regression model enhanced with Polynomial Features for capturing non-linear relationships.

Interactive UI: Modern, responsive web interface built with Bootstrap 5.

FastAPI Backend: High-performance Python API for handling real-time predictions.

Dockerized: Fully containerized using Docker for seamless deployment on platforms like Railway.
